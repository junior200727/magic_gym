# magic_gym

Projeto PHP simples para gerenciamento (arquivos existentes do diretório `magic_gym`).

Como usar (localmente):

- Coloque os arquivos dentro da pasta do seu servidor (ex.: XAMPP: `htdocs/magic_gym`).
- Configure o arquivo `db.php` com as credenciais do seu banco de dados.
- Acesse via browser: `http://localhost/magic_gym`.

Passos para publicar no GitHub:

1. Crie um repositório no GitHub (nome sugerido: `magic_gym`).
2. Adicione o remote e rode:

```bash
git remote add origin https://github.com/SEU_USUARIO/NOME_REPO.git
git branch -M main
git push -u origin main
```

Ou instale e use o GitHub CLI (`gh`) para criar o repositório automaticamente.
